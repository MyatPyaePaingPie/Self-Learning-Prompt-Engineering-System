# Paing Week 3 - Streamlit Component

Simple form with text input and output box.

## How to run:

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run streamlit_app.py
```

3. Open your browser to http://localhost:8501

## What it does:
- User enters text in the input field
- Output box displays "Processed: [input text]"
- Simple and functional as requested for Week 3
