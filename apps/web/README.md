# Web Interface

Streamlit-based web interface for the Self-Learning Prompt Engineering System.

## Prerequisites

1. **Backend API must be running** on http://localhost:8000
2. **Python 3.11+** with virtual environment activated
3. **Groq API Key** set up in backend (see backend README)

## Setup

### Install Dependencies

```bash
cd apps/web
pip install -r requirements.txt
```

## Running the Web Interface

Make sure the backend is running first, then:

```bash
cd /path/to/Self-Learning-Prompt-Engineering-System/apps/web
streamlit run streamlit_app.py
```

The web interface will be available at: http://localhost:8501

## What It Does

- **Prompt Input**: Enter any prompt you want to improve
- **Improve Prompt Button**: Sends your prompt to the backend API
- **Improved Output**: Displays the enhanced version generated by Groq's LLM
- **Error Handling**: Shows helpful error messages if something goes wrong

## Troubleshooting

**Error: "Connection refused" or API errors**
- Make sure the backend is running on http://localhost:8000
- Check that your `.env` file has `GROQ_API_KEY` set up

**Streamlit won't start**
- Activate virtual environment: `source venv/bin/activate`
- Reinstall dependencies: `pip install -r requirements.txt`

**Blank page or won't load**
- Check terminal for error messages
- Make sure port 8501 is not already in use
